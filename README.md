# GTPlaytest
Playtest Gorilla Tag maps with Lua support.
# WARNING
To playtest in VR, Windows or Linux is required. You can import a WASD function to use WASD movement on macOS or non-VR mode. And also the project doesn't come with GT movement. You have to import that yourself, GT movement can be found at https://github.com/Another-Axiom/GorillaLocomotion.
